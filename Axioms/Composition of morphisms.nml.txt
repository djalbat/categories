Variable A:Object
Variable B:Object
Variable C:Object
Variable D:Object

Variable f:Morphism
Variable g:Morphism
Variable h:Morphism

Axiom (CompositionOfMorphisms)
  Suppose 
    f:A ⟶ B
    g:B ⟶ C
  Then
    f;g:A ⟶ C

Axiom (AssociativityOfComposition)
  Suppose 
    f:A ⟶ B
    g:B ⟶ C
    h:C ⟶ D
  Then
    (f;g);h = f;(g;h) = f;g;h
